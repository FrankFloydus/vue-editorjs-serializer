# vue-editorjs-serializer

A small tool to serialize [Editor.js](https://editorjs.io/) JSON output.
It provides basic serialization of native blocks.

- header
- paragraph
- image
- list
- raw html
- code
- blockquote
- links
- tables

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

npm run serve
