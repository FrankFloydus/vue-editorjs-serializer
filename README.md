# vue-editorjs-serializer

A VueJS component to serialize [Editor.js](https://editorjs.io/) JSON output.
It provides basic serialization of native blocks:

- Header
- Paragraph
- Image
- List
- Raw HTML
- Code
- Blockquote
- Links
- Tables

## Usage

```
<Vserializer />
```

| prop | type   | value               |     |
| ---- | ------ | ------------------- | --- |
| json | Object | EditorJS Json Input |     |
